2. Install Java JDK 17 or 21
3. Add Java to PATH:

====================================================

TROUBLESHOOTING
=================

Issue: "javac is not recognized"
Solution: Use full path to javac.exe as shown above

Issue: GUI doesn't open
Solution: Ensure you're using the correct Java path, or install JDK

Issue: Application runs but no text displays
Solution: The terminal might not support Unicode - this is normal

====================================================

APPLICATION FEATURES SUMMARY
============================

MODULE 1: Smart City Route Planner
- Add/Remove locations and roads
- Display all connections (graph visualization)
- BST In-Order traversal (sorted locations)
- BST Tree structure display
- BFS Traversal (Queue-based)
- DFS Traversal (Stack-based)
- Shortest path finding (BFS-based)

MODULE 2: Data Sorter
- Manual data entry (comma-separated numbers)
- Random data generation
- Bubble Sort implementation
- Merge Sort implementation  
- Quick Sort implementation
- Performance comparison table

MODULE 3: Algorithm Performance Analyzer
- Linear Search analysis
- Binary Search analysis
- Bubble/Merge/Quick Sort analysis
- Multiple input sizes (100, 500, 1000, 2000, 5000)
- Full performance report
- Time complexity summary
